# Customer Churn AI Agent Notebooks

This folder is designed to store Jupyter Notebooks used for:
1. **Exploratory Data Analysis (EDA):** Detailed distribution, balance check, and demographic correlations with churn.
2. **Model Research and Development:** Trialing different models, hyperparameters, and custom loss functions.
3. **Explainability Experiments:** Custom SHAP / LIME attribution plots.

You can place your `.ipynb` notebooks here to keep them organized!
